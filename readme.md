## Brownies

This website shows you how to make delicious brownies based on the youtube video by preppy kitchen! In this website, I used HTML, CSS and a small amount of JS for the timer. In the HTML, I mostly used `<a href=>` tags and `<ul` for the lists. This site helped e to experiment with the different HTML elements that I can add and rely on `<strong>` and `<small>` without explicity relying on CSS to do it for me. 

Overall, this website turned out to look very aesthetic with the brown colour palette and in the future I want to add more animations to the site and interactivity!
